# Save Our Wildlife

This is a simple web page made for a multimedia CAT assignment. It was built
using separate HTML, CSS, and JS files like we were taught in class.

## Files

- `index.html` - the main page (heading, video, description, button)
- `styles.css` - makes the page look nice and responsive
- `script.js` - makes the button play or hide the video
- `wildlife.mp4` - the video file (you need to add this yourself)

## What the page does

- Shows a heading that says "Save Our Wildlife"
- Has a video with play/pause/volume controls
- Has a short paragraph explaining what the video is about
- Has a button that plays the video if it's paused, or hides it if it's playing
- Looks okay on both a computer and a phone screen

## How to run it

1. Put `index.html`, `styles.css`, `script.js`, and `wildlife.mp4` in the same folder.
2. Open `index.html` in a browser.

## How to put it on GitHub Pages

1. Make a new public repo on GitHub.
2. Upload these files to the repo (don't forget wildlife.mp4).
3. Go to Settings > Pages, choose the main branch, and save.
4. The site will be live at:
   `https://your-username.github.io/your-repo-name/`
